## Overview

**Tower of Blocks** is a visually engaging stacking game built to test precision, rhythm, and reaction timing. The objective is simple: stack moving blocks as accurately as possible and build the tallest tower without losing stability.

## Gameplay

- Time each block drop carefully
- Stack blocks with maximum precision
- Keep the tower balanced as it grows
- Aim for the highest score possible
- Challenge your focus and consistency with every move

## Features

- Smooth and addictive gameplay
- Clean and modern game interface
- Precision-based block stacking
- Score-focused progression
- Lightweight and fast browser performance
- Beginner-friendly project structure
